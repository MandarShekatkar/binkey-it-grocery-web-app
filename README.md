# Binkey IT – Real-Time Grocery Delivery Web Application

A full-stack grocery delivery web application built using the MERN stack with secure authentication, payment integration, and order workflows.

## Features
- JWT-based authentication with email/OTP verification
- Product listing, cart, and order management
- Stripe payment integration
- Admin and user role-based access
- Responsive UI with Redux state management

## Tech Stack
- Frontend: React.js, Redux, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Payments: Stripe

## Project Structure
/client – React frontend  
/server – Node.js & Express backend

## Note
This project is built for learning and demonstration purposes.
